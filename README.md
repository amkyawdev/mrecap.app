# Movie Recap App 🎬

A React-based web application for creating video recaps with subtitles and voiceover. Features include video trimming, SRT subtitle editing, audio mixing, and PWA support.

## Features

- 📹 Video selection and trimming
- 📝 SRT subtitle editor with live preview
- 🎙️ Audio upload and voice recording
- 📤 Export final video
- 📱 PWA - Installable web app
- 🔴 Red theme (background: #FF0000)

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Zustand (state management)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Project Structure

```
src/
```
